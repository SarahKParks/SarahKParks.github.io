My Work

Smart Tic-Tac-Toe

Constructing robotic tic-tac-toe board plays against a user and responds to voice commands by the user to move their tic-tac-toe pieces. Processing voice commands on web app with Python backend and implementing JavaScript API for incorporating speech recognition into web page. Implemented tic-tac-toe game using minimax algorithm.

USC's Rocket Propulsion Lab

Creating an interface that automatically tracks details about mechanical parts for the avionics team using McMaster-Carr’s API with a Python backend to improve avionics order processes’ efficiency by 50%. Implementing a software defined radio using a HackRF One to measure power inputs on sensors and receive and demodulate raw GPS data to test the rocket’s GPS. Converting template GeckoDrive program into a template G-code program for X-Winder filament winding machine.

Texas A&M Motion Capture Lab

Reconstructed data using MATLAB to visually validate the motion of points over time after calibrating the MCS using the direct linear transformation method. Tested and modified existing main OpenCV Python program to improve robustness by measuring point values over time, resulting in successful identification of tracking errors over 60% of the time. Project received 1st Place Overall in Texas A&M’s Virtual Project Showcase for the Fall 2019 semester.
